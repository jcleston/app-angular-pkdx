Para iniciar o desenvolvimento, é necessário efetuar as seguintes instalações: NPM, Node e AngularCli

## Comandos
Para iniciar o serviço
```shell
$ ng s -o
```

Para criar um Component
```shell
$ ng g c nome_component
```
